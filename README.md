# Portfolio Website

A modern portfolio website built with React + Vite, featuring:

- Interactive UI with Framer Motion animations
- Three.js powered background effects
- Responsive design with modern CSS
- Dark mode support
- Component-based architecture

## Tech Stack

- React
- Vite
- Framer Motion
- Three.js
- CSS Modules

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

- Hero section with animated background
- About section with professional experience
- Projects showcase
- Skills overview
- Contact form
- Responsive navigation
- Smooth scrolling
- Modern UI/UX
